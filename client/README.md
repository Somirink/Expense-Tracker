## 💰 Expense Tracker App
---
This project is a full-stack Expense Tracker application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It provides a simple yet efficient Income & Expense Management System, allowing users to securely manage their transactions, track total balance, and view category-wise analytics through an interactive dashboard.

The application supports user authentication and ensures that each user can manage their own financial data independently.
---

## 🔧 Technologies Used
---

- **Frontend:** React.js, React Router, Recharts, CSS  

- **Backend:** Node.js, Express.js  

- **Database:** MongoDB with Mongoose  

- **Authentication:** JWT (JSON Web Token), bcrypt.js

---

## 📋 Features
---
🔐 Authentication

- **Register User:** Create a new account securely.
- **Login User:** Authenticate using email and password.
- **Logout:** Securely log out from the application.
- **User Display:** Displays the logged-in user's name dynamically.

💵 Transaction Management

- **Add Transaction:** Add income or expense with amount, category, date, reference, and description.
- **Delete Transaction:** Remove a transaction permanently.
- **View Transactions:** View all personal transactions in table format.

🔎 Filtering

- **Filter by Type:** View only Income or Expense.
- **Filter by Frequency:** Filter transactions by Week, Month, or Year.

📊 Analytics Dashboard

- **Total Transactions:** Displays total number of transactions.
- **Total Income:** Shows total income amount.
- **Total Expense:** Shows total expense amount.
- **Total Balance:** Displays remaining balance.
- **Category-wise Income Chart:** Visual breakdown of income by category.
- **Category-wise Expense Chart:** Visual breakdown of expenses by category.
- **Toggle View:** Switch between Table view and Analytics view.
